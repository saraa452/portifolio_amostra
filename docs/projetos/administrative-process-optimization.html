    Methodology

    How this administrative project was structured

    The analysis maps administrative workflows from request to decision, highlighting cycle-time patterns, conversion rates and risk concentration.

    1. Process mapping

    Consolidated each process by status, department, contract type and approval owner.

    2. KPI design

    Built indicators for approval rate, value flow, pending load and operational bottlenecks.

    3. Monitoring layer

    Delivered a static dashboard for governance routines and strategic review meetings.

    Stack

    Tools used in the project

    • Chart.js for visualization of workflow and performance indicators
    • HTML, CSS and JavaScript for interactive reporting
